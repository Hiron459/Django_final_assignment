# StockFlow（備品貸出・在庫管理アプリ）

学校や職場にある備品の在庫と貸出状況を管理するDjangoアプリです。

## 主な機能

- 備品の登録（名前、カテゴリ、所持数、保管場所、備考）
- 備品の一覧表示・キーワード検索
- 利用可能数と在庫僅少の表示
- 貸出登録と返却処理
- 貸出中・返却済みの履歴表示、返却期限超過の警告

## 使用技術

- Python 3.14
- Django 6.0
- SQLite

## 起動方法

```bash
.venv/bin/python manage.py migrate
.venv/bin/python manage.py runserver
```

ブラウザで `http://127.0.0.1:8000/` を開きます。

## 課題要件との対応

- Model: `Item`（備品）と `Loan`（貸出記録）を定義
- View: 一覧、登録、返却の処理を実装
- Template: 共通レイアウトを継承した画面を実装
- レイアウト: ダッシュボード、在庫警告、検索、モバイル対応を用意
